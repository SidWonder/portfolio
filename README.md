# Портфолио

## Описание

Данный проект представляет собой сайт-портфолио, написанный на React.
В данном сайте я использую хуки, css-модули, grid/flex.

## Как запустить сайт
Сайт задеплоен на GitHub pages - https://sidwonder.github.io/portfolio/
